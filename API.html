DLNIRSP_Pipeline.Data Module¶

Classes¶

AlignmentData([data_dir, dataset_ID, memmap])

AlignmentData is an extension of FitData designed to only hold data from the Alignment InstrumentProgramTask (DKIST004).

CSStep(header)

This class allows for an easy way to quickly compare SPEC-0122 style headers to determine if they come from the same Calibration Sequence (CS) step.

CalibrationCurveData([CS_step, memmap])

CalibrationCurveData is an extension of FitsData that loads a single Calibration Curve from a directory containing a full Set of Calibration Curves (SoCC).

ConfigParser([defaults, dict_type, …])

ConfigParser implementing interpolation.

DarkData([data_dir, dataset_ID, memmap])

DarkData is an extension of FitData designed to only hold data from the Dark InstrumentProgramTask (DKIST004).

DataSetData([data_dir, dataset_ID, suffix])

DataSetData performs 2 functions: First, it separates DL mosaic/repeat numbers (MRN) found among raw data in a directory.

DemodulationData()

DemodulationData is a small modification of the FitsData class.

FitsData([load_obj, memmap, allow3D])

FitsData is a basically a wrapper for astropy.io.fits, with some extra methods and properties to aid the reduction of DKIST data.

GainData([data_dir, dataset_ID, memmap])

GainData is an extension of FitData designed to only hold data from the Gain InstrumentProgramTask (DKIST004).

MRNData(MRN[, memmap])

MRNData is an extension of FitsData that loads a single DL-NIRSP mosaic/repeat number from a directory containing a full DL-NIRSP DataSet.

MosiacRepeatNumber(header)

This class allows for an easy way to quickly compare SPEC-0122 style headers to determine if they come from the same DL-NIRSP mosaic/repeat number (MRN).

ObservationData([data_dir, dataset_ID, memmap])

ObservationData is an extension of FitData designed to only hold data from the Observation InstrumentProgramTask (DKIST004).

SoCCData([data_dir, suffix])

SoCCData performs 2 functions: First, it separates and sorts Calibration Sequence (CS) steps found among raw data in a directory.

Time(val[, val2, format, scale, precision, …])

Represent and manipulate times and dates for astronomy.
