DataSetData¶

class DLNIRSP_Pipeline.Data.DataSetData(data_dir=None, dataset_ID='', suffix='*.fits')[source]¶

Bases: object

DataSetData performs 2 functions: First, it separates DL mosaic/repeat numbers (MRN) found among raw data in a directory. Secondly, it can initialize and produce an MRNData object at a requested MRN.

Initialize properties and maybe scan a directory for mosaic/repeat steps.

Parameters
data_dirstr

Path to a directory that will be searched for MRNs

suffixstr

File mask. The full mask will be data_dir/*SUFFIX

read_directory(self, data_dir, suffix='*.fits', dataset_ID='')[source]¶

Read all files in a directory and assign all files a MRN identifier. Then populate a dictionary that maps mosaic/repeat number to a list of files corresponding to that MRN.

Parameters
data_dirstr

Path to a directory that will be searched for CS steps

suffixstr

File mask. The full mask will be data_dir/*SUFFIX

dataset_IDstr

Only check data from this data set. A check is made against each file’s ID_004 header value. Can be a blank string to just load all data sets.
