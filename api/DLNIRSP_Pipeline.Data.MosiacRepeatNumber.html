MosiacRepeatNumber¶

class DLNIRSP_Pipeline.Data.MosiacRepeatNumber(header)[source]¶

Bases: object

This class allows for an easy way to quickly compare SPEC-0122 style headers to determine if they come from the same DL-NIRSP mosaic/repeat number (MRN). The smallest possible DL data unit is defined uniquely by a) its mosaic location in the FOV, and b) its repeat number. For example, if the Instrument Program requires a 2x2 mosaic of the FOV to be repeated 3 times then there will be 12 unique data units. Each data unit is defined by the vector:

(DSPSRepeatNumber, CurrentSpatialStepX, CurrentSpatialStepY)

This class is that vector.

This class can be sorted and is hashable for use in dictionaries.

Initialize and read the MRN from a SPEC-0122 FITS header.
