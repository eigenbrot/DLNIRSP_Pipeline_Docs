find_spec_geometry¶

DLNIRSP_Pipeline.SolarFlatCalibration.find_spec_geometry(data, binary, peak_width=4, max_shift=20, debug=False)[source]¶

Find the spectral shifts/curvature for a single slit

From the perspective of this software there is no difference between a shift caused by spectrograph optics (curvature) and a shift caused by any other reason. The 1D shift for every spatial position is computed relative to the center position of the slit.

Shifts are determined by correlating each spectrum with the centeral spectrum. The peak in the correlation function is then interpolated to find the final shift.

Once all shifts have been found they are globally shifted such that the average shift is 0.

Parameters
datanumpy.ndarray

Data array of a single slit. Should be corrected for any rotations such that the dispersion axis is parallel to the pixel axis.

binarynumpy.ndarray

A “bad-pixel-mask” corresponding to the input data. Pixels that should be ignored are 1 and all else are 0. Spatial positions with an average mask value > 0.4 will not be shifted.

peak_widthint

The width of the pixel window to use when interpolating the peak of the correlation function. Passed directly to peakutils.interpolate’s width parameter.

max_shiftfloat

The maximum allowable shift, in data pixel units. Any shifts larger than this will be ignored and that spatial position will be given the previously found shift.

debugbool

If True plot the shifts as a function of slit position. Useful for looking at slit functions or whatever.

Returns
numpy.ndarray

1D array containing a pixel shift for every spatial pixel.
