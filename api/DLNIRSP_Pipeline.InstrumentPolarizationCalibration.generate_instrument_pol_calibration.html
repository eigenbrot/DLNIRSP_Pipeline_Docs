generate_instrument_pol_calibration¶

DLNIRSP_Pipeline.InstrumentPolarizationCalibration.generate_instrument_pol_calibration(dmodData, geo_cal, output)[source]¶

Read raw demodulation matrices and prep them for use in Science data reduction

The demodulation matrices for each DL-NIRSP beam are treated separately and if either one was a full-frame average then the spatial dimensions are simply squeezed out. Otherwise the data are passed to upsample_demodulation for interpolation to the full beam size.

Parameters
dmodDataData.DemodulationData

The demodulation matrix for both DL beams. Should have a single data HDU for each beam with shape (x, y, l, 4, M) where (x, y, l) is the shape of the binned DL-NIRSP beams frame passed to PA&C and M is the number of unique modulator states.

geo_caldict

Polarimetric geocal object generated by gen_inst_pol_geo_cal. The mask from this is used to mask out the bad regions in the final demodulation matrices.

outputstr

Location to save the final demodulation matrices

Returns
Data.DemodulationData

Object containing a data HDU for each DL-NIRSP beam. The data shape of these HDUs will either be (X, Y, L, 4, M) or (4, M), where X and Y are the full read-out size of a single DL-NIRSP beam and L is the number of slits (5).
