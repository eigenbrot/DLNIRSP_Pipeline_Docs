find_px_angles¶

DLNIRSP_Pipeline.SolarFlatCalibration.find_px_angles(H, theta, debug=False)[source]¶

Find the most significant angles in a Hough transform.

Peaks in the Hough transform a first found with a simple max filter and then refined via interpolation of the surrounding peak.

Note, peak angles are not allowed to be within pi/10 of each other.

Parameters
Hnumpy.ndarray

A 2D array representing a Hough accumulator

thetanumpy.ndarray

A 1D array of the angle values corresponding to H

debugbool

If True a plot showing the peak angles and interpolated values will be saved as “peaks.png” in the current directory.

Returns
numpy.ndarray

The most significant angels found
