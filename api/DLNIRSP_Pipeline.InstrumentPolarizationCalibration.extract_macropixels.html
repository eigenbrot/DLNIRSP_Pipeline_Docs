extract_macropixels¶

DLNIRSP_Pipeline.InstrumentPolarizationCalibration.extract_macropixels(fitsData, borders)[source]¶

Extract macropixels and conform data for use by the PAC Modules

Currently this function simply averages each slitbeam into a single pixel. Because PAC expects 3D data the final form for each beam is (1, 1, S), where S is the number of DL slits.

In the future more advanced macropixilization will result in data with an arbitrary shape (X, Y, S).

Parameters
fitsDataData.FitsData

Data to extract macropixels from. Should be a single, reduced CS step.

bordersnumpy.ndarray

Pixel indices of the borders between each slitbeam

Returns
Data.FitsData

Data from Beam 1 suitable for use by the PAC Modules

Data.FitsData

Data from Beam 2 suitable for use by the PAC Modules
