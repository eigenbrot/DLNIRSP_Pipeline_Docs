main¶

DLNIRSP_Pipeline.ScienceCalibration.main(datasetData, MRN, outpre, dark_cal=None, instrument_pol_cal=None, solar_flat_cal=None, geometric_cal=None, dark_cal_sig=None, solar_flat_cal_sig=None, allow_no_pol=False, ifu_cal=None, cube_shape=(100, 100), telescope_db=None, remove_TM=True, include_M12=True, remove_parallactic=True)[source]¶

Reduce DLNIRSP Science data

Any intermediate Calibration objects that are provided will be applied to the data and are technically optional. If no Calibration objects are provided then the data will essentially be passed through, which is pretty boring. An slight exception to this is the instrument_pol_cal; this is psuedo-required and a user must prove their sanity via the allow_no_pol keyword to turn off demodulation.

Returns
Data.FitsData

Reduced science data. If IFU resampling is done (i.e., Level 1.5 data) there will be four 3D data_hdus, corresponding to the four Stokes vectors. If no ifu_cal is provided then the output will be Level 1 data, with each of the four data_hdus containing all DL fibers stacked into a single slit.
